O git e o github sao um sistema de cotrole de versao que funciona no seu computador(git) e na nuvem(github).
O git tira "fotos" do seu codigo a cada mudanca,permitido voltar no tempo ou testar ideais sem quebrar o original.
Ogithub guarda tudo isso na internet,funcionando como backup e rede social para programadores,varias pessoas podem mexer no mesmo projeto sem baguncar o trabalho umas das outras,e voce ainda pode mostrar seuas projetos como portifolio.
